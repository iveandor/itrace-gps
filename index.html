자녀 갤럭시 워치용

iLog 갤럭시 워치

  • GPS 실시간 위치 추적
  • 버튼 하나로 SOS 긴급 발신
  • 배터리 절약 모드 (야간·안심존)
  • 실내외 자동 감지
자녀 스마트폰용

iLogMobile 스마트폰

자녀가 가진 스마트폰을 그대로 추적기로 사용합니다. 별도 기기가 필요 없습니다.

  • 화면이 꺼져도 위치 자동 추적
  • SOS 발신 + 현장 영상(Android) or 음성(iPhone) 녹화
  • 안심존 진입·이탈 자동 감지
  • 위치 이력 자동 저장
  • 실내외 자동 감지

  • 이동 경로 이력 조회

  • 여러 자녀·여러 보호자 함께 모니터링

핵심 기능

강력한 기능, 쉬운 사용

복잡한 설정 없이 가족 모두가 즉시 사용할 수 있습니다.

SOS 긴급 알림

버튼 하나로 즉시 보호자에게 알림. 위치가 자동 전송되고, 현장 녹음·녹화는 Basic 요금제 이상에서 제공됩니다(Android 자녀폰은 영상+음성, iPhone·갤럭시 워치는 음성).

알림 전 요금제 · 녹화 Basic 이상

실시간 위치 추적

스마트 GPS 보정 기술로 실내외 모두에서 정확한 위치를 제공합니다. 위성·하이브리드 지도 전환과 즉시 위치 확인 요청도 지원합니다.

전 요금제

안심존 설정

학교, 집 등 안전 구역을 반경 40~500m 슬라이더로 세밀하게 지정하고(지도에 실시간 원형 미리보기), 진입·이탈 시 즉시 알림을 받습니다.

전 요금제

배터리 최적화

야간 절약 모드, 안심존 모드로 배터리를 아끼며 하루 종일 추적합니다. 배터리 잔량에 따라 GPS 정밀도도 자동 조절됩니다.

전 요금제

이동 경로 이력

날짜별 이동 경로를 번호 마커·경로선으로 확인합니다. 보관 기간은 무료 7일, 유료 요금제는 30일입니다.

전 요금제 · 보관 기간 요금제별

이력 누락 방지

통신이 잠시 끊겨도 그동안의 위치가 기기에 임시 저장되어, 연결이 복구되면 자동으로 동기화됩니다.

전 요금제

원격 음성 모니터링

아이가 걱정될 때 자녀 스마트폰 주변의 소리를 30초 단위로 원격 청취합니다(음성 전용). 반드시 자녀 동의 하에 사용하세요.

Standard 요금제 이상

방문장소 분석

하루 동안 아이가 오래 머문 장소(체류지점)를 자동으로 분석해 카드와 지도로 요약합니다.

Standard 요금제 이상

기기 찾기

아이와 100m 이내로 가까워지면 실시간 거리와 방향 화살표를 안내하고, 소리 울리기로 마지막 몇 걸음까지 찾아드립니다.

전 요금제

예상시각 미도착 알림

학교·학원 같은 안심존에 도착 예정 시각과 요일을 설정해 두면, 마감 시각까지 도착하지 않을 때 즉시 알려드립니다. 등하교 누락을 가장 먼저 알 수 있습니다.

전 요금제

알림 맞춤 설정

자녀마다 안심존·도착·배터리·추적 중단 알림을 따로 켜고 끄고, 방해금지 시간대를 지정합니다. SOS 긴급 알림만은 항상 켜져 절대 놓치지 않습니다.

전 요금제

내 데이터 직접 관리

날짜별 위치 이력, SOS 세션과 녹화, 모니터링 기록을 회원 탈퇴 없이 직접 영구 삭제할 수 있어 프라이버시에 안심입니다.
