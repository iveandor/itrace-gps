1앱 소개 및 역할

서비스는 세 가지 앱으로 구성됩니다. 보호자가 iTrace(보호자용)로 자녀의 위치를 모니터링하고, 자녀는 iLog(갤럭시 워치) 또는 iLogMobile(스마트폰)을 사용해 위치를 전송합니다.

2iTrace 회원가입

iTrace

보호자용 앱에서 처음 시작하는 단계입니다.

  1. iTrace 앱을 실행합니다. 시작 화면에서 "회원가입" 버튼을 탭합니다.
  2. 약관 동의 화면이 표시됩니다. 아래 2가지 항목에 모두 체크한 뒤 "전체 동의" 버튼을 탭합니다.
    • 이용약관 (필수)
    • 개인정보 처리방침 (필수)
  3. Google 계정 선택 화면이 열립니다. 사용할 Google 계정을 선택합니다.
  4. 회원가입이 완료되어 메인 화면(지도)으로 이동합니다.
이미 계정이 있으신가요?

"회원가입" 대신 "로그인" 버튼을 탭하세요. 아래 섹션을 참고하세요.

3iTrace 로그인

iTrace
  1. iTrace 앱 시작 화면에서 "로그인" 버튼을 탭합니다.
  2. Google 계정 선택 화면에서 가입 시 사용한 계정을 선택합니다.
  3. 계정 확인 후 메인 화면으로 이동합니다.
로그인 오류 — "회원가입을 하지 않았습니다."

선택한 Google 계정으로 가입된 iTrace 계정이 없을 때 표시됩니다.
해당 계정으로 먼저 회원가입을 진행하거나, 다른 Google 계정으로 다시 시도하세요.

다른 기기에서 로그인하면?

iTrace는 계정당 활성 기기 1대만 지원합니다.
새 기기에서 로그인하면 이전 기기에서 자동으로 로그아웃됩니다.

4iLog (Galaxy Watch) 연동

iTrace iLog

갤럭시 워치에 iLog가 설치된 경우 아래 순서로 연동합니다.

사전 준비

Galaxy Watch에 iLog 앱이 설치되어 있어야 합니다. Galaxy Store에서 "iLog" 검색 후 설치하세요.

보호자 (iTrace) 측 — 연동 코드 생성

  1. iTrace 앱 하단 탭에서 "설정"을 탭합니다.
  2. "자녀 앱 연동" 또는 "새 기기 추가"를 탭합니다.
  3. 화면에 6자리 영숫자 코드가 표시됩니다 (예: A3KR7X). 이 코드의 유효 시간은 24시간입니다.
  4. 코드를 자녀에게 알려주고, 자녀가 Galaxy Watch에서 코드를 입력할 때까지 화면을 유지합니다.

자녀 (iLog · Galaxy Watch) 측 — 코드 입력

  1. Galaxy Watch에서 iLog 앱을 실행합니다.
  2. 처음 실행 시 연동 코드 입력 화면이 표시됩니다. 워치 화면의 키보드 또는 크라운을 이용해 보호자가 알려준 6자리 코드를 입력합니다.
  3. 입력 완료 시 자동으로 연동이 처리됩니다. "연동 완료" 메시지가 표시되면 성공입니다.
  4. 이후 iLog가 자동으로 위치 추적을 시작합니다.

iTrace에서 연동 확인

코드 생성 화면에서 연동 완료 알림이 표시되거나, 설정 → 자녀 앱 목록에 새 워치가 나타나면 연동 성공입니다.

코드 입력 오류
  • "유효하지 않은 코드" — 코드가 잘못 입력되었거나 만료(24시간 초과)되었습니다. iTrace에서 새 코드를 생성하세요.
  • "이미 사용된 코드" — 해당 코드는 이미 다른 기기에서 사용되었습니다. 새 코드를 생성하세요.
  • "다른 계정에 연동된 기기" — 해당 워치가 다른 보호자 계정에 이미 연동되어 있습니다. 먼저 기존 계정에서 연동 해제 후 재시도하세요.

5iLogMobile (스마트폰) 연동

iTrace iLogMobile

자녀 스마트폰에 iLogMobile이 설치된 경우 아래 순서로 연동합니다.

사전 준비

자녀 스마트폰에 iLogMobile 앱을 설치하세요 (Play Store / App Store 검색).
앱 실행 후 Google 계정으로 로그인이 필요합니다.

보호자 (iTrace) 측 — 연동 코드 생성

iLog 연동과 동일합니다 (iTrace → 설정 → 자녀 앱 연동 → 6자리 코드 확인).

자녀 (iLogMobile) 측 — 코드 입력

  1. iLogMobile 앱을 실행하고 Google 계정으로 로그인합니다.
  2. 로그인 후 연동 코드 입력 화면이 표시됩니다. 보호자에게 받은 6자리 코드를 입력합니다.
  3. "연동하기" 버튼을 탭합니다.
  4. 연동 성공 시 메인 화면으로 이동하고, 위치 추적이 자동으로 시작됩니다.
권한 요청 안내

처음 연동 후 위치, 알림, 배터리 최적화 제외 등 여러 권한 요청 화면이 나타납니다.
정상적인 추적을 위해 모든 권한을 허용해 주세요.

위치 추적 중단 및 재개 (자녀 직접 제어)

자녀는 iLogMobile 앱 설정에서 언제든지 위치 추적을 중단하거나 재개할 수 있습니다.

  1. iLogMobile 앱 → 설정 탭을 탭합니다.
  2. "위치 추적" 토글을 탭하여 중단하거나 재개합니다.
  3. 추적을 중단하면 GPS 수집이 즉시 멈추고, 연동된 보호자에게 "위치 추적 중단" 알림이 전송됩니다.

6위치 추적

iTrace에서 자녀 위치 확인

  1. 메인 화면(지도)에서 연동된 자녀의 아이콘을 확인합니다.
  2. 아이콘을 탭하면 최근 위치 시각, 배터리 잔량, 활동 상태가 표시됩니다.
    "로드뷰" 버튼을 탭하면 해당 위치의 거리뷰를 확인할 수 있습니다 (Kakao Map 지원 지역 한정).
  3. "위치 확인하기" 버튼을 탭하면 자녀 기기에 즉시 위치 수집 요청을 보냅니다 (원할 때 즉시 확인).

벨소리 울리기

자녀의 기기를 원격으로 울려 위치를 확인하게 할 수 있습니다.

  1. iTrace 지도 화면에서 자녀 기기 아이콘을 탭합니다.
  2. "벨소리" 버튼을 탭합니다.
  3. 자녀의 기기(스마트폰 또는 Galaxy Watch)에서 벨소리가 울려, 주변에서 기기 위치를 찾는 데 도움을 줍니다.
벨소리는 모든 자녀 기기에서 사용 가능

벨소리 기능은 자녀의 iLogMobile(스마트폰)과 Galaxy Watch(iLog) 모두에서 동작합니다.

기기 찾기 (근처에서 정확히 찾기)

자녀(또는 자녀의 기기)와 가까운 거리에 있을 때, 마지막 몇 걸음을 정확히 좁혀 찾도록 도와줍니다.

  1. 지도에서 보호자(나)와 자녀 기기가 약 100m 이내로 가까워지면 화면에 "기기 찾기" 카드가 자동으로 나타납니다.
  2. 카드에 실시간 거리(미터)와 방향 화살표가 표시됩니다(방향은 지도 기준 북쪽 기준이며 나침반이 아닙니다).
  3. 15m 이내로 들어오면 "근처" 모드로 전환되어 매우 가까이 있음을 안내합니다.
  4. 카드의 "소리 울리기" 버튼으로 기기를 울리고, "위치 새로고침"으로 최신 위치를 다시 받아 마지막 위치를 좁힙니다.
거리·방향은 GPS 근사값입니다

기기 찾기는 GPS 위치를 기반으로 한 근사 안내로, 100~30m 구간에서 가장 유용합니다. 아주 가까운 실내·근접 거리에서는 소리 울리기로 마무리하세요.

추적이 멈춘 것 같다면?

자녀 기기의 배터리 절약 모드, 백그라운드 제한, 인터넷 연결 상태를 확인하세요.
iTrace 설정 → 해당 기기 → 진단 정보에서 마지막 업로드 시각을 확인할 수 있습니다.

7안심존

특정 장소(학교, 집 등)에 가상 경계선을 설정하고, 자녀가 진입하거나 이탈할 때 알림을 받습니다.

안심존 추가

  1. iTrace 앱에서 지도에서 안심존을 설정할 자녀를 선택합니다.
  2. "안심존" 탭 또는 지도 내 메뉴에서 "안심존 추가"를 탭합니다.
  3. 지도에서 중심 위치를 직접 지정하거나 주소를 검색합니다.
  4. 반경을 40~500m 슬라이더로 세밀하게 조정합니다(지도에 원형 미리보기가 실시간으로 표시됩니다). 진입 알림 / 이탈 알림 여부를 선택합니다.
  5. 장소 이름을 입력하고 저장합니다.

알림 방식

이벤트알림 대상내용
안심존 진입보호자 (iTrace)"[자녀 이름]이 [장소명] 영역에 진입했습니다."
안심존 이탈보호자 (iTrace)"[자녀 이름]이 [장소명] 영역에서 이탈했습니다."

예상시각 미도착 알림

학교·학원처럼 정해진 시각까지 도착해야 하는 안심존에는 도착 예정 시각을 설정해 둘 수 있습니다. 마감 시각까지 자녀가 해당 안심존에 도착하지 않으면 보호자에게 즉시 알림이 전송되어, 등하교 누락을 가장 먼저 알 수 있습니다.

  1. 안심존을 추가하거나 기존 안심존을 수정하는 화면에서 "예상 도착 시각" 항목을 켭니다.
  2. 도착해야 하는 시각과 적용할 요일(예: 평일)을 선택합니다.
  3. 마감 시각까지 도착이 확인되지 않으면 "[자녀 이름]이 [장소명]에 예정 시각까지 도착하지 않았습니다" 알림이 발송됩니다.
도착이 확인되면 알림은 발송되지 않습니다

마감 시각 이전에 자녀가 안심존에 진입하면 정상 도착으로 처리되어 미도착 알림이 오지 않습니다.

안심존 알림이 오지 않을 때

iTrace 알림 권한이 허용되어 있는지 확인하세요. 기기 설정 → 앱 → iTrace → 알림 → 허용.

8SOS 긴급 알림

자녀가 위험 상황에서 SOS를 발신하면 보호자에게 즉시 긴급 알림이 전송됩니다.

SOS 발신 방법

기기발신 방법
iLog Galaxy Watch iLog 앱 화면에서 SOS 버튼을 2초 이상 길게 누르기
화면에 원형 게이지가 채워지며 발동됩니다.
iLogMobile 스마트폰 iLogMobile 메인 화면에서 SOS 버튼을 2초 이상 길게 누르기
화면에 게이지 링이 채워지며 발동됩니다.

SOS 발신 시 동작

  • GPS 위치 수집 주기가 10초로 강제 전환
  • iTrace 앱에 빨간색 긴급 알림 배너 표시 + 해당 위치로 지도 자동 이동
  • iLog (Galaxy Watch): 주변 음성이 자동 녹음되어 iTrace에서 재생 가능
  • iLogMobile: 주변 영상(Android 전용) + 음성이 자동 녹화되어 iTrace에서 재생 가능
  • SOS는 발신 후 3분 후 자동 해제 (또는 보호자가 iTrace에서 수동 해제 가능)
  • 사이렌(경보음)은 SOS와 별개의 독립 기능입니다 — SOS 발신만으로 자동 재생되지는 않으며, 자녀 기기(iLog·iLogMobile)의 앱 내 사이렌 버튼으로 직접 켜거나 보호자가 보내는 '벨소리'로 울릴 수 있습니다.
  • 오프라인 상태에서 발신된 SOS는 네트워크 복구 즉시 자동 전송됩니다 (SOS 대기열 보존)

9원격 모니터링 (스트리밍)

iTrace iLog iLogMobile

iLog(갤럭시 워치) 또는 Android 자녀폰(iLogMobile)이 연동된 경우, 보호자가 자녀 기기의 주변 오디오(음성)를 원격으로 모니터링할 수 있습니다.

주의 — 동의 필수

원격 모니터링 기능은 반드시 자녀의 동의 하에 사용해야 합니다. 무단 도청은 법적으로 금지됩니다.

사용 방법

  1. iTrace 앱에서 모니터링할 자녀 기기를 선택합니다.
  2. "원격 모니터링" 버튼을 탭합니다.
  3. 자녀 기기에 요청이 전송되고, 30초 단위로 녹음된 오디오가 iTrace에서 재생 가능합니다.
  4. 종료하려면 "중지" 버튼을 탭합니다.
iPhone(iLogMobile)은 원격 모니터링 불가

원격 모니터링은 iLog(Galaxy Watch)와 Android 자녀폰(iLogMobile)에서 지원됩니다. iPhone(iLogMobile)에서는 지원되지 않습니다.

10배터리 절약 설정

자녀 기기의 배터리 소모를 줄이기 위해 시간대별 GPS 수집 주기를 조정할 수 있습니다.

설정 방법

  1. iTrace 앱 → 설정 → 해당 자녀 기기 선택
  2. "배터리 절약"을 탭합니다.
  3. 야간 절약 시간대(예: 오후 11시 ~ 오전 7시)를 설정하고, 해당 시간대의 수집 주기를 조정합니다.
  4. 저장하면 변경 사항이 자녀 기기에 즉시 반영됩니다.

자동 배터리 보호

설정과 무관하게 자녀 기기 배터리 잔량에 따라 자동으로 GPS 정밀도를 조절합니다.

배터리 잔량GPS 모드
20% 초과고정밀 모드 (정확도 우선)
5 ~ 20%균형 모드 (정확도·배터리 균형)
5% 이하절전 모드 (배터리 우선)
배터리 부족 알림

자녀 기기 배터리가 일정 수준 이하로 떨어지면 보호자에게 푸시 알림이 발송됩니다.

11위치 이력

iTrace에서 자녀의 과거 이동 경로를 날짜별로 확인할 수 있습니다.

  1. iTrace 메인 화면에서 "이력" 탭을 탭합니다.
  2. 조회할 자녀를 선택하고 날짜를 선택합니다.
  3. 지도에 해당 날짜의 위치 기록이 번호 마커로 순서대로 표시됩니다.
  4. 번호 마커를 탭하면 해당 시각의 위치 정보를 확인할 수 있습니다.
이력 보관 기간

위치 이력은 수집 시점으로부터 최대 30일(요금제에 따라 다름) 후 자동으로 삭제됩니다.

12디바이스 관리 및 해지

연동된 기기 목록 확인

  1. iTrace 앱 → 설정 → "연동된 기기" 탭
  2. 현재 연동된 iLog / iLogMobile 기기 목록이 표시됩니다.
  3. 각 기기의 마지막 접속 시각, 배터리, 앱 유형을 확인할 수 있습니다.

기기 이름 변경

  1. 기기 목록에서 이름을 변경할 기기를 탭합니다.
  2. "이름 수정"을 탭하고 새 이름을 입력 후 저장합니다.

연동 해지

  1. 기기 목록에서 해지할 기기를 탭합니다.
  2. "연동 해지" 버튼을 탭합니다.
  3. 확인 다이얼로그에서 "해지"를 탭합니다.
  4. 해지 시 해당 기기의 모든 위치 이력, 안심존 데이터가 삭제되며, 자녀 기기에 연동 해제 알림이 전송됩니다.
연동 해지는 되돌릴 수 없습니다

해지 후 모든 데이터가 영구 삭제됩니다. 재연동이 필요한 경우 새 코드를 생성하여 처음부터 다시 연동해야 합니다.

자녀 기기에서 연동 해제됐을 때

보호자가 연동을 해지하면 자녀의 iLog / iLogMobile 앱에 알림이 표시되고, 앱은 자동으로 연동 코드 입력 화면으로 이동합니다. 이후 재연동 전까지 위치 추적이 중단됩니다.

13자주 묻는 질문

구독·결제, 권한, 위치 업데이트 등 자주 받는 질문을 모았습니다.

Q 서비스는 유료인가요? 어떻게 구독하나요?

신규 가입 시 7일간 모든 기능(원격 모니터링·30일 이력·방문장소 분석·SOS 녹화)을 결제 정보 입력 없이 무료로 체험할 수 있습니다. 영구 무료 플랜은 없으며, 체험이 끝나면 Basic·Standard·Family 중 하나를 구독해야 계속 이용할 수 있습니다.

유료 구독은 iTrace 앱 실행 후 구독 화면에서 플랜을 선택하고 App Store 또는 Google Play를 통해 결제합니다. 요금 및 플랜 상세는 iTrace 앱 내 구독 화면 및 각 스토어 제품 페이지에서 확인할 수 있습니다.

Q 구독을 취소하려면 어떻게 하나요?

구독은 결제한 플랫폼에서 직접 취소합니다.

  • iOS — 설정 → Apple 계정 → 구독 → iTrace 선택 → 구독 취소
  • Android — Google Play → 프로필 아이콘 → 구독 → iTrace 선택 → 구독 취소

자동 갱신을 방지하려면 현재 구독 기간 만료 24시간 전까지 취소해야 합니다.

Q 자녀 기기가 여러 대인데 모두 연동할 수 있나요?

네. iTrace 하나의 계정에 여러 기기(iLog + iLogMobile 혼합)를 연동할 수 있습니다. 각 기기마다 독립된 위치 이력과 안심존이 관리됩니다. 단, 구독 플랜에 따라 연동 가능한 기기 수 제한이 있을 수 있습니다.

Q 자녀 기기를 교체했을 때 어떻게 하나요?

기존 기기는 설정에서 연동 해지한 후, 새 기기에 iLog 또는 iLogMobile을 설치하고 새 코드로 재연동하세요.

Q iLog (Galaxy Watch)를 재설치하면 연동이 사라지나요?

iLog는 앱 재설치 후에도 동일 워치·계정에서 실행하면 이전 연동 정보를 자동으로 복원합니다. 복원이 실패할 경우에만 코드 입력 화면이 표시됩니다.

Q 위치가 업데이트되지 않아요.

아래 순서로 확인하세요.

  1. 자녀 기기의 인터넷(Wi-Fi 또는 모바일 데이터) 연결 상태를 확인합니다.
  2. iLog / iLogMobile 앱이 백그라운드에서 실행 중인지 확인합니다.
  3. 배터리 최적화 예외 설정이 되어 있는지 확인합니다.
  4. iTrace에서 "위치 확인하기" 버튼으로 즉시 수집을 요청합니다.
Q 야간에 위치 업데이트가 너무 뜸해요.

배터리 절약 설정에서 야간 시간대 수집 주기를 단축하거나, 야간 절약 모드를 비활성화하세요 (iTrace → 설정 → 해당 기기 → 배터리 절약). iLogMobile은 야간에 정밀 위치를 자동으로 더 자주 수집합니다.

Q 앱이 정상적으로 동작하지 않아요 (iLogMobile / iLog).

앱 내 진단 화면에서 문제를 확인하고 자동으로 수정할 수 있습니다.

  • iLogMobile — 메인 화면 → 설정 → 진단 → 문제 항목 옆 "수정" 버튼 탭. 서비스 재시작이 필요한 경우 자동으로 처리됩니다.
  • iLog — 설정 화면 → 진단 → 문제 항목 탭. 권한 재요청 또는 배터리 최적화 예외 설정을 안내합니다.
Q SOS 알림을 받지 못했어요.

iTrace 앱의 알림 권한을 확인하고, 앱을 완전히 종료했다가 다시 실행해 보세요. 그래도 문제가 지속되면 기기 설정 → 앱 → iTrace → 알림 → 허용 여부를 확인하세요.

Q 보호자 계정을 삭제하고 싶어요.

iTrace 앱 → 설정 → 계정 → "계정 삭제"를 통해 탈퇴할 수 있습니다. 탈퇴 시 모든 연동 기기, 위치 이력, 안심존 데이터가 영구 삭제됩니다.

Q 알림을 자녀별로, 종류별로 끄거나 켤 수 있나요?

네. iTrace 알림 설정에서 자녀마다 안심존·예상시각 미도착·배터리·추적 중단 알림을 각각 켜고 끌 수 있습니다. 또한 방해금지 시간대(예: 취침 시간)를 지정해 해당 시간에는 알림을 받지 않을 수 있습니다.

단, SOS 긴급 알림은 안전을 위해 항상 켜져 있으며, 방해금지 시간에도 예외 없이 전달됩니다.

Q 위치 이력이나 SOS 기록만 따로 삭제할 수 있나요?

네. 회원 탈퇴까지 가지 않아도 민감한 기록을 직접 관리할 수 있습니다.

  • 위치 이력 — 이력 화면에서 날짜를 선택한 뒤 삭제 버튼으로 해당 날짜 기록을 영구 삭제합니다(기기 소유자만 가능).
  • SOS 세션·녹화 — SOS 기록 목록에서 개별 세션을 삭제하면 연결된 음성·영상도 함께 삭제됩니다.
  • 원격 모니터링 기록 — 모니터링 세션도 개별 삭제할 수 있습니다.

삭제된 데이터는 복구할 수 없습니다.
