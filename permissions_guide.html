문제가 발생하면 아래 가이드의 설정을 다시 확인해 주세요.

자녀 앱 iLogMobile Android

iLogMobile은 백그라운드에서 지속적으로 위치를 수집하는 앱입니다. 아래 설정이 빠지면 화면이 꺼진 후 위치 업데이트가 중단됩니다.

위치 권한 — 항상 허용

  1. 기기 설정 앱을 엽니다.
  2. 앱 (또는 애플리케이션 관리자) 을 선택합니다.
  3. iLog Mobile 을 찾아 탭합니다.
  4. 권한 › 위치 를 선택합니다.
  5. 항상 허용을 선택합니다. ("앱 사용 중에만"은 백그라운드 추적이 중단됩니다.)

배터리 최적화 제외

  1. 기기 설정 앱을 엽니다.
  2. 앱 › iLog Mobile 을 선택합니다.
  3. 배터리 를 탭합니다.
  4. 제한 없음을 선택합니다. (기본값 "최적화"는 위치 수집을 방해합니다.)

알림 권한 (Android 13 이상)

  1. 설정 › 앱 › iLog Mobile › 알림 을 선택합니다.
  2. 알림 허용을 활성화합니다.

신체 활동 인식 권한 (Android 10 이상)

활동 상태(이동/정지) 감지 및 배터리 절약 기능에 필요합니다.

  1. 설정 › 앱 › iLog Mobile › 권한 을 선택합니다.
  2. 신체 활동 (또는 활동 인식) › 허용을 선택합니다.

마이크 권한 (SOS 음성 녹음 · 원격 모니터링)

SOS 발생 시 자동 음성 녹음 및 보호자의 원격 오디오 모니터링 기능에 필요합니다.

  1. 설정 › 앱 › iLog Mobile › 권한 을 선택합니다.
  2. 마이크 › 허용을 선택합니다.
OEM별 추가 설정이 필요할 수 있습니다

위 설정 후에도 백그라운드 추적이 끊기면, 기기 제조사별 추가 설정을 확인해 주세요.

자녀 앱 iLogMobile iOS

위치 권한 — 항상

  1. iPhone 설정 앱을 엽니다.
  2. 앱 목록에서 iLog Mobile 을 탭합니다.
  3. 위치 를 탭합니다.
  4. 항상을 선택합니다. ("앱 사용 중"을 선택하면 화면이 꺼질 때 GPS가 중단됩니다.)
  5. 정확한 위치 토글을 켭니다.

백그라운드 앱 새로 고침

  1. 설정 › iLog Mobile 을 탭합니다.
  2. 백그라운드 앱 새로 고침을 활성화합니다.

마이크 권한 (SOS 음성 녹음 · 원격 모니터링)

SOS 발생 시 자동 음성 녹음 및 보호자의 원격 오디오 모니터링 기능에 필요합니다.

  1. 설정 › iLog Mobile › 마이크 를 탭합니다.
  2. 허용을 선택합니다.
저전력 모드 주의

저전력 모드(설정 › 배터리 › 저전력 모드)가 켜져 있으면 백그라운드 앱 새로 고침이 자동으로 비활성화됩니다. 추적이 필요한 시간에는 저전력 모드를 끄거나 수동으로 다시 활성화해 주세요.

자녀 앱 iLog Wear OS

iLog는 Android Watch에서 동작하며, 워치와 페어링된 스마트폰의 Wear OS 앱을 통해 권한을 관리합니다.

위치 권한 — 항상 허용

  1. 워치에서 설정 › 앱 › iLog 를 탭합니다.
  2. 권한 › 위치 › 항상 허용을 선택합니다.

배터리 최적화 제외 (Wear OS 3 이상)

  1. 워치에서 설정 › 앱 › iLog › 배터리 를 탭합니다.
  2. 배터리 최적화 해제를 선택합니다.
앱 내 진단 화면 활용

iLog 앱 → 설정 → 진단 화면에서 배터리 최적화 항목이 표시되면 해당 항목을 탭하여 자동으로 설정 화면으로 이동할 수 있습니다.

보호자 앱 iTrace Android

iTrace는 보호자가 지도를 보는 앱입니다. 백그라운드에서도 SOS 알림과 안심존 알림을 받으려면 아래 설정이 필요합니다.

위치 권한 — 앱 사용 중 허용

  1. 설정 › 앱 › iTrace › 권한 › 위치 를 탭합니다.
  2. 앱 사용 중에만 허용 이상을 선택합니다.

알림 권한

  1. 설정 › 앱 › iTrace › 알림 을 탭합니다.
  2. 알림 허용을 활성화합니다. (SOS · 안심존 알림 수신에 필수)

보호자 앱 iTrace iOS

위치 권한 — 앱 사용 중

  1. 설정 › iTrace › 위치 를 탭합니다.
  2. 앱 사용 중 이상을 선택합니다.

알림 권한

  1. 설정 › iTrace › 알림 을 탭합니다.
  2. 알림 허용을 활성화합니다.
  3. SOS 알림이 즉시 울리도록 소리 및 배너도 함께 활성화하는 것을 권장합니다.
